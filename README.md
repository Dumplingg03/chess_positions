# Chess SDC Generator

## Описание проекта

Chess SDC Generator — интеллектуальная система генерации шахматных задач с использованием условной генеративно-состязательной сети (CGAN) и шахматного движка Stockfish.

Система автоматически создаёт шахматные позиции заданного уровня сложности, анализирует их и оценивает при помощи собственной метрики сложности — **SDC (Strategic Difficulty Coefficient)**.

Проект сочетает:

- генеративные нейросети;
- анализ шахматных позиций;
- автоматическую оценку сложности;
- фильтрацию задач;
- систему аналитики и логирования через ClearML.

---

# Основные возможности

## Генерация шахматных позиций

CGAN создаёт новые шахматные позиции на основе:

- случайного латентного вектора;
- заданного рейтинга сложности.

---

## Анализ через Stockfish

После генерации позиции анализируются движком Stockfish:

- поиск лучшего хода;
- анализ тактики;
- оценка позиции;
- вычисление principal variation.

---

## Автоматическая оценка сложности (SDC)

Система рассчитывает сложность позиции с помощью собственной формулы, основанной на:

- длине тактической линии;
- разнице между лучшим и вторым ходом;
- количестве легальных ходов.

---

## Система фильтрации

Некорректные или слабые позиции автоматически отбрасываются.

Проверяются:

- наличие королей;
- материальный баланс;
- минимальное количество фигур;
- наличие тактической идеи;
- существование лучшего хода.

---

## Аналитика и визуализация

Через ClearML система автоматически строит:

- графики рейтингов;
- ошибки модели;
- производительность;
- статистику генерации;
- внутренние характеристики задач.

# Механизм генерации задач

## 1. Выбор целевого рейтинга

Система выбирает целевой рейтинг сложности:

```python
target_rating = 1500
```

---

## 2. Нормализация рейтинга

Для подачи в нейросеть рейтинг нормализуется:

```python
r_norm = (target_rating - 500) / 2500
```

Это переводит рейтинг в диапазон `[0 ; 1]`.

---

## 3. Генерация позиции

CGAN получает:

- случайный шум `z`;
- условие сложности `r_norm`.

```python
output = model(z, r_norm)
```

---

## 4. Восстановление шахматной доски

Выходной тензор преобразуется в полноценную шахматную позицию:

```python
board = get_board_from_output(output)
```

---

## 5. Проверка корректности

Проверяются:

- наличие королей;
- количество фигур;
- материальный баланс.

---

## 6. Анализ позиции через Stockfish

Stockfish выполняет:

- поиск лучшего хода;
- анализ тактики;
- оценку вариантов;
- построение principal variation.

---

## 7. Расчёт SDC

После анализа вычисляется коэффициент сложности позиции.

---

# Формула SDC

## Основная формула

```python
m4 = (real_moves * 3.9) + (np.log1p(gap) * 0.5) - (mobility * 0.1)
```

---

## Перевод в рейтинговую шкалу

```python
sdc_rating = (m4 * 85.0) + 220
```

---

# Параметры SDC

## Gap

Разница между лучшим и вторым лучшим ходом.

```python
gap = abs(best_move - second_move)
```

### Что показывает

- высокий Gap → есть один сильный ход;
- низкий Gap → несколько похожих вариантов.

Большой Gap характерен для тактических задач.

---

## Mobility

Количество легальных ходов в позиции.

```python
mobility = board.legal_moves.count()
```

### Что показывает

- высокое значение → позиция более вариативная;
- низкое значение → позиция более форсированная.

---

## Real Moves

Длина лучшей линии ходов (Principal Variation).

```python
real_moves = len(pv)
```

### Что показывает

Количество ходов, необходимых для реализации лучшей комбинации.

---

# Проверка соответствия рейтингу

После вычисления SDC выполняется сравнение:

```python
abs(sdc - target_rating) <= 200
```

Если разница небольшая — позиция принимается.
