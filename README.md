# Chess Puzzle Generator (CVAE + Stockfish)

**Первая демоверсия** системы генерации шахматных задач, использующая вариационный автоэнкодер (CVAE) для создания позиций и движок Stockfish для тактической проверки. Проект выполнен в рамках Лабораторной работы №4.

## Описание проекта
Данный инструмент автоматизирует процесс создания шахматных головоломок. Вместо случайного перебора фигур, нейросеть обучается на паттернах реальных партий, что позволяет генерировать более "человекоподобные" и естественные расстановки. После генерации каждая позиция проходит через фильтр движка Stockfish, чтобы убедиться в наличии уникального и тактически верного решения.



## Стек технологий
* **Python 3.10+**
* **PyTorch** — разработка и обучение архитектуры CVAE.
* **python-chess** — обработка шахматной логики и формата FEN.
* **Pygame** — графический интерфейс пользователя (GUI).
* **Stockfish Engine** — внешний движок для оценки сложности и валидации задач.


Основные возможности:
- Генерация через AI: Использование обученной модели CVAE для предсказания позиций.
- Тактическая фильтрация: Проверка позиций на наличие форсированных матов или выигрыша материала.
- Визуализация: Интерактивная шахматная доска на базе Pygame.
- Гибкая настройка: Возможность изменять глубину анализа и параметры генерации.
